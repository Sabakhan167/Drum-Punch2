# Drump Punch Landing Page

Welcome to the **Drump Punch** landing page!  
This is a responsive, modern landing page for the Drump Punch game, built with HTML and CSS.

## 🚀 Features

- **Fully Responsive:** Looks great on desktop, tablet, and mobile devices.
- **Animated Scroller:** Eye-catching, animated "PUNCH 2 EARN" ticker.
- **Modern Layout:** Clean sections for About, How to Play, Journey, FAQ, and more.
- **Custom Design:** Matches Figma design specs, including grid backgrounds and styled buttons.
- **Social Links:** Stylish Telegram and Twitter/X buttons.
- **Easy to Customize:** All styles in `styles.css`.
